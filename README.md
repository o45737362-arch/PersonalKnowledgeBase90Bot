# Personal Knowledge Base Bot

A Telegram bot for storing, organizing, and retrieving your personal knowledge.

## Features
- 📝 Save notes with titles and content
- 🔍 Search through your knowledge base
- 🏷️ Organize with tags (#work, #personal, #code)
- 📊 Track your learning stats
- 📤 Export your data anytime
- 🔄 Update and delete notes
- 📚 Never lose your ideas again!

## Commands
- `/add Title | Content` - Save a new note
- `/search keyword` - Search your notes
- `/list` - List all notes
- `/tag #tagname` - Get notes by tag
- `/get 123` - Get a specific note
- `/update 123 | new content` - Update a note
- `/delete 123` - Delete a note
- `/stats` - View your stats
- `/export` - Export all notes
- `/help` - Show help
